Digital Tools & Resources

A comprehensive collection of digital humanities tools, tutorials, and resources for scholars, students, and anyone interested in computational approaches to humanistic inquiry.

Navigating the Digital Humanities Toolkit

The digital humanities landscape offers a vast array of tools and methodologies. This curated collection represents tools I regularly use in research and teaching, organized by functionality and accompanied by practical guidance.

📊 Text Analysis

Voyant Tools
Web-based text analysis environment for digital humanities research. Perfect for beginners and exploratory analysis.
Visit Tool Tutorial
MALLET
Machine learning toolkit for topic modeling, document classification, and sequence tagging. Command-line based but powerful.
Download Guide
spaCy
Industrial-strength natural language processing library for Python. Excellent for entity recognition and linguistic analysis.
Documentation Examples
AntConc
Free concordance software for corpus linguistics research. Great for keyword analysis and text pattern identification.
Download Tutorials

📈 Data Visualization

D3.js
JavaScript library for creating dynamic, interactive data visualizations. Steep learning curve but unlimited possibilities.
Official Site Gallery
Tableau Public
Free version of Tableau for creating and sharing interactive visualizations online. User-friendly with drag-and-drop interface.
Get Started Examples
Gephi
Open-source network analysis and visualization software. Essential for exploring relationships and connections in data.
Download Tutorials
Palladio
Web-based platform for visualizing complex historical data. Designed specifically for humanities researchers.
Launch App Help

📚 Digital Publishing

Omeka S
Web publishing platform for digital collections and exhibitions. Perfect for creating online archives and scholarly projects.
Learn More Examples
Scalar
Authoring and publishing platform for multimodal scholarship. Supports complex, non-linear narratives.
Platform User Guide
TEI Publisher
Publishing framework for TEI-encoded texts. Professional-grade digital edition publishing.
Demo Documentation
GitBook
Modern documentation and knowledge management platform. Great for collaborative writing and publishing.
Try It Features

GitHub
Version control and collaboration platform. Essential for managing code, datasets, and collaborative digital projects.

Zotero
Research management tool for collecting, organizing, and sharing research sources. Excellent group collaboration features.
Download Quick Start
Hypothesis
Social annotation tool for collaborative reading and discussion. Add conversations directly to web pages and documents.

Team communication platform. Useful for organizing research groups and maintaining ongoing project discussions.

Jupyter Notebooks

Learning Pathways

New to digital humanities? These curated learning paths will help you develop skills progressively, from beginner to advanced practitioner.

🌱 Beginner Path

Start your digital humanities journey with user-friendly tools that require no programming experience.
  1. Voyant Tools (text analysis)
  2. Tableau Public (visualization)
  3. StoryMapJS (storytelling)
  4. Omeka (digital collections)
Start Learning

🚀 Intermediate Path

Build on foundational skills with tools that offer more customization and analytical power.
  1. R and RStudio basics
  2. MALLET topic modeling
  3. Gephi network analysis
  4. GitHub for collaboration
Continue Journey

⚡ Advanced Path

Develop programming skills and work with cutting-edge computational methods in humanities research.
  1. Python for text analysis
  2. Machine learning with scikit-learn
  3. D3.js for custom visualizations
  4. Neural networks for NLP
Advance Skills
